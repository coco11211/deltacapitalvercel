Accessibility Statement

Delta Capital is committed to ensuring digital accessibility for all visitors, including those with disabilities.

Our Commitment

We are dedicated to making our website accessible to everyone, regardless of ability. We strive to conform to WCAG 2.1 Level AA accessibility standards and continuously work to improve the accessibility of our digital properties.

Accessibility Features

Delta Capital implements the following accessibility features:

  • Semantic HTML: Our website uses proper semantic markup for document structure, enabling screen readers and assistive technologies to correctly interpret content.
  • Color Contrast: We maintain sufficient color contrast ratios (WCAG AA minimum 4.5:1 for normal text) to ensure readability for users with low vision or color blindness.
  • Keyboard Navigation: All interactive elements are fully accessible via keyboard navigation. Users can navigate through links, buttons, and form fields using Tab and Enter keys.
  • Focus Indicators: Visual focus indicators are provided for keyboard navigation, clearly showing which element currently has focus.
  • Alt Text: All informative images include descriptive alt text. Decorative images are properly marked as such.
  • Responsive Design: Our website is fully responsive and works well on devices of all sizes, from mobile phones to large desktop displays.
  • Text Scaling: The website supports text size adjustment up to 200% without loss of functionality or readability.

Known Limitations

We are aware of the following accessibility limitations and are actively working to address them:

  • Data Visualizations: Some complex data visualizations and charts may not be fully accessible to screen reader users. We provide alternative text descriptions where possible and are working to implement ARIA data tables for improved accessibility.
  • PDF Documents: Older PDF documents may not be fully tagged for accessibility. New documents are created with accessibility in mind.

Third-Party Content

Our website includes content and services from third parties, such as embedded fonts from Google Fonts. We work with service providers to ensure they meet accessibility standards where possible, but we are not responsible for the accessibility practices of third-party content.

Browser and Assistive Technology Compatibility

Our website is compatible with the following browsers and assistive technologies:

  • Browsers: Chrome, Firefox, Safari, and Edge (latest versions)
  • Screen Readers: NVDA, JAWS, and VoiceOver
  • Other: Speech recognition software, magnification tools, and other assistive technologies

Assessment and Testing

We regularly assess our website for accessibility compliance using:

  • Automated testing tools (WAVE, Axe, Lighthouse)
  • Manual testing with keyboard navigation
  • Screen reader testing (NVDA, JAWS)
  • User testing with people with disabilities

Feedback and Support

We welcome feedback on the accessibility of our website. If you encounter any accessibility barriers or have suggestions for improvement, please contact us:
