How to Use These Email Signatures

  1. Click "Copy HTML" button to copy the signature code
  2. Gmail: Go to Settings → Forwarding and POP/IMAP → Signature. Paste into the signature field.
  3. Outlook: Go to File → Options → Mail → Signatures. Click "New" and paste the HTML code.
  4. Apple Mail: Go to Mail → Preferences → Signatures. Create new signature and paste.
  5. Important: Make sure your email client supports HTML signatures. All major providers do.
